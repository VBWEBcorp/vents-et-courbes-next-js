# Guide d'Administration - Vents et Courbes

## Accès à l'administration

**URL:** `/admin/login`

Connectez-vous avec vos identifiants pour accéder au tableau de bord d'administration.

## Menu de Navigation

Le menu latéral vous permet d'accéder à toutes les sections :

- **Dashboard** : Vue d'ensemble avec statistiques
- **Stages** : Gérer les stages de céramique
- **Cours** : Gérer les cours réguliers
- **Blog** : Gérer les articles de blog
- **Pages** : Modifier les textes du site

## Gestion des Contenus de Pages

### Qu'est-ce que c'est ?

La section "Pages" vous permet de modifier tous les textes qui apparaissent sur votre site web, sans avoir besoin de toucher au code. Chaque élément de texte (titre, sous-titre, description, bouton) peut être modifié facilement.

### Comment modifier un contenu ?

1. Cliquez sur "Pages" dans le menu latéral
2. Utilisez le filtre pour sélectionner une section (Accueil, À propos, Contact, etc.)
3. Trouvez le contenu que vous souhaitez modifier
4. Cliquez sur l'icône de crayon (✏️) à droite
5. Modifiez le texte dans le formulaire
6. Cliquez sur "Enregistrer"

Les modifications sont immédiatement visibles sur le site !

### Organisation du contenu

Les contenus sont organisés par section :

#### 🏠 Accueil (Home)
- Titre principal du Hero
- Sous-titre du Hero
- Description
- Textes des boutons d'action
- Titre et description de la section "À propos"

#### ℹ️ À propos (About)
- Titre et sous-titre de la page
- Texte de présentation
- Description de l'équipe

#### 🎓 Cours
- Titre et sous-titre de la page
- Description des cours

#### 📅 Stages
- Titre et sous-titre de la page
- Description des stages

#### 💼 Formation Pro
- Titre et sous-titre
- Description de la formation professionnelle
- Informations sur le CAP

#### 📧 Contact
- Titre et sous-titre
- Adresse
- Email
- Téléphone

#### 📝 Blog
- Titre et sous-titre de la page blog
- Description

### Conseils pratiques

✅ **Bonnes pratiques:**
- Gardez les titres courts et impactants
- Assurez-vous que les textes sont clairs et sans fautes
- Testez vos modifications en visitant le site public
- Les textes des boutons doivent être des appels à l'action clairs

⚠️ **Important:**
- Vous ne pouvez pas créer ou supprimer de contenus, seulement les modifier
- Chaque contenu a un rôle spécifique sur le site
- Les modifications sont instantanées et visibles par tous

## Gestion des Stages

1. Cliquez sur "Stages" dans le menu
2. Vous pouvez :
   - **Ajouter** un nouveau stage
   - **Modifier** un stage existant
   - **Supprimer** un stage
3. Remplissez tous les champs requis (titre, durée, niveau, image, description, etc.)
4. Sauvegardez vos modifications

## Gestion des Cours

Fonctionne de la même manière que les stages. Permet de gérer les cours réguliers (annuels, trimestriels, etc.).

## Gestion du Blog

1. Cliquez sur "Blog" dans le menu
2. Vous pouvez créer, modifier ou supprimer des articles
3. Chaque article nécessite :
   - Un titre
   - Un slug (URL de l'article)
   - Un auteur
   - Une date de publication
   - Une image
   - Le contenu de l'article
4. Activez/désactivez la publication avec le bouton "Actif"

## Navigation

- Utilisez le menu latéral pour naviguer entre les sections
- Sur mobile, cliquez sur le menu burger (☰) pour accéder au menu
- Cliquez sur "Voir le site →" en bas du menu pour ouvrir le site public
- Utilisez "Déconnexion" pour vous déconnecter

## Support

Pour toute question ou problème, contactez votre administrateur système.
